## 🛠️ Tech Stack & Core Architecture
* **Cloud Infrastructure:** AWS (EC2, Security Groups, VPC)
* **Containerization:** Docker & Docker Compose
* **CI/CD Automation:** GitHub Actions (Automated syntax verification, integration checks, and secure SSH deployments)
* **Backend & Database:** Python Application with MongoDB state management
* **Reverse Proxy & Web Server:** Nginx Configured as a reverse proxy
